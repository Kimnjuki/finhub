# Cryptocurrency Trading Dashboard

A comprehensive trading dashboard with real-time cryptocurrency data, forex markets, stock markets, AI predictions, and advanced charting tools.

## Features

- **Real-time Crypto Prices**: Live cryptocurrency prices and market data
- **Forex Trading**: Major currency pairs with live rates
- **Stock Markets**: Fortune 500 companies and trading signals
- **AI Predictions**: Machine learning-powered price predictions
- **Advanced Charting**: TradingView-style charts with technical indicators
- **Event Calendar**: Economic events and market-moving news
- **Portfolio Tracking**: Track your trading portfolio performance
- **Risk Calculator**: Calculate position sizes and risk management
- **Trading Simulator**: Practice trading with virtual money

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (Backend)
- Recharts (Charts)
- shadcn/ui (UI Components)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
